About DLSUD Campus Navigation System

DLSUD Campus Navigation System
By The Three Musketeers – Discrete Structures I

Overview

Welcome to the DLSUD Campus Navigation System! This prototype demonstrates how Discrete Structures can solve real-world campus problems. Using Graph Theory, Propositional Logic, and Set Theory, the system provides tools for navigation, scholarship evaluation, and parking management.

Features

Campus Navigation

  • Computes the shortest walking paths between major campus buildings using Dijkstra’s Algorithm.
  • Highlights the route on the interactive campus map.
  • Helps new students and visitors find their way efficiently.

Scholarship Eligibility Checker

  • Evaluates your qualifications for academic, financial, and athletic scholarships.
  • Uses propositional logic to ensure fair and consistent results.
  • Shows which requirements are met and which are missing.

Campus Parking Allocation

  • Tracks parking slot availability in east and west lots.
  • Uses set operations to update occupancy and free slots in real time.
  • Simplifies parking decisions and saves time.

How to Use

  • Select the feature you want to use from the main menu.
  • Follow on-screen prompts to input buildings, scholarship info, or parking slots.
  • The system will display results clearly and visually, e.g., shortest paths or available slots.

Notes & Limitations

  • This is a prototype; only selected buildings and parking areas are included.
  • Features demonstrate educational applications of Discrete Structures rather than full-scale campus services.
